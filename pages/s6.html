// Session 5-7 : Design and Evaluation.

--Introduction.

This section involves the development of the core project as a group and culmination at the evaluation of the project in session 7.

--The Prototype.

The initial prototyping sketch simply included the drawing of the units generating the audio plugging into an output module.


Fig : The sketch of the music production cards.

--Implementation.

The cards were implemented as breadboard implementations of various audio components such as amplifier blocks that accept an audio signal and processes it by applying an amplification process to it.


Fig : Amplifier module assembled in a breadboard circuit.

As for the input module, we developed a music selector module using a standard MP3 player module that contains samples of various tones of a piano. Along with that we used a OLED display that can showcase the current tone that is playing and lets the user select and pick the tone they want and add that to the output effectively creating a tone picking circuit.

The fidelity and interactability of this has been further improved by use of a RGB sensor that can be used to sense a color coded block that can represent an instrument which can be used for tone picking.


Fig : tone picker circuit.
